Node es un framework que te permite ejecutar archivos de javascript, por lo que todos los archivos que creemos serán
.js.
En vez de ejecutar las cosas en el navegador, como estamos acostumbrados, las ejecutaremos en nuestro sistema operativo.
Como si fuera un programa como otro cualquiera.
Para ejecutarlo entramos en la consola, nos movemos a la carpeta donde esté el archivo que queremos ejecutar
y ponemos node nombreArchivo.js
